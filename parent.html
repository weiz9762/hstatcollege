Financial Aid

  1. In order to receive Federal Financial Aid, parents should make a FAFSA account to create a FAFSA ID.
  2. After your child completes the FAFSA, encourage them to apply for New York State Financial Aid(TAP) if they are attending a school in New York State by making a NYSTAP account.
  3. Encourage your child to Apply For Scholarships by accessing the HSTAT shared Google Drive Folder.
  4. Many, not all, schools require a CSS Profile in addition to the FAFSA. There is a list of schools that require the CSS profile that can be accessed here. If the school your child is applying to requires a CSS Profile, apply for the CSS Profile through this specfic College Board Site.
